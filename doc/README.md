# Fs-mini

Легковесный и быстрый ИИ-ассистент для Windows, работающий поверх всех окон. Использует локальные LLM (через Ollama), умеет искать свежую информацию в сети (DuckDuckGo) и распознавать текст на экране.

## Установка
1. Установите [Ollama](https://ollama.com/) и скачайте нужную модель.
2. Установите [Tesseract OCR](https://github.com/UB-Mannheim/tesseract/wiki) (с поддержкой русского языка) и укажите путь к нему в `config.py`.
3. Установите зависимости Python:
   ```bash
   pip install pillow pytesseract duckduckgo-search requests keyboard
   ```
#Использование
Shift + Space — Открыть/скрыть строку ввода.

Alt + Shift + Space — Выделить область экрана для OCR и перевода.

Начните запрос с /s  (например: /s погода Москва), чтобы ассистент нашел ответ в интернете.

Введите /quit для выгрузки программы из памяти.
